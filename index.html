Summary: Latitude vs Aspects of Weather

The purpose of this project was to deteremine the correlation, if any, between the temperature, wind, humidity and cloudiness and the distance from the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

Once we assembled the dataset, we used Matplotlib to plot the varios aspects of the weather with respect to latitude. This site provides the source data and visualizations for each aspect of the weather as well as the conslusion of our findings.
